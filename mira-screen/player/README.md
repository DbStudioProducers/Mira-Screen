# Mira Player

Player de vídeo próprio e independente da Mira Screen.

### Features
- Video.js base + Shaka Player
- Suporte a HLS/DASH
- DRM Widevine + FairPlay
- Quality selector
- Analytics em tempo real
- Watermark dinâmico

### Desenvolvimento

```bash
npm install
npm run dev
```

Abra `index.html` para demo.
